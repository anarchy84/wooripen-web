### 1-2. 공통 톤앤매너

- **어미**: "~해요" 체 (친근한 존댓말) — 3사 공통
- **문장 길이**: 15~25자 이내 짧은 문장
- **핵심 키워드**: "모든", "쉽고", "빠른", "편리한", "깔끔한"
- **피하는 표현**: 기술 용어 직접 노출 없음 (PG, VAN 등은 맥락 속에서만)
- **감성 포인트**: 문제해결형 ("고민 끝!", "따로 부탁하지 않아도")

### 1-3. 우리편 적용 카피 전략

**핵심 포지셔닝**: "매장 운영의 모든 것, 한 번에 해결"

**추천 메인 카피안**:
1. "인터넷부터 단말기·CCTV까지, 매장의 모든 인프라를 한 곳에서"
2. "사장님의 매장 고민, 우리편이 해결해드려요"
3. "통신·결제·보안, 따로 찾지 마세요. 우리편 하나면 끝"

**CTA 전략**:
- 1차 CTA: "무료 상담 받기" (토스 스타일, 부담 제거)
- 2차 CTA: "견적 받아보기" (구체적 액션)
- 플로팅: 카카오톡 상담 + 전화 (현행 유지)

**가격 앵커링 적용**:
- "설치비 0원" + "월 비용 ₩XX,XXX~" 형태
- "3사 비교견적으로 최대 40% 절감" 같은 비교 우위

**신뢰 수치 적용**:
- "47,200+ 매장이 선택한" (이미 적용됨, 유지)
- "상담 후 평균 3일 내 설치 완료"
- "고객 만족도 4.87/5.0"

## 2. 개발팀 보고 — 디자인 / 레이아웃 / 동적 표현

### 2-1. 토스플레이스 디자인 분석

- **히어로**: 풀스크린 제품 사진 배경 + 오버레이 텍스트
- **제품 쇼케이스**: 화이트/블랙 2컬러 제품을 나란히 배치, 배경 그라데이션
- **프로모션 바**: 정가 취소선 + 할인율 강조 (큰 폰트, 컬러 대비)
- **CTA 플로팅**: 하단 고정 바, 항상 노출
- **특징**: 매우 큰 제품 이미지, 텍스트 최소화, 시각 중심

### 2-2. 토스페이먼츠 디자인 분석

- **레이아웃**: 풀페이지 스크롤 (섹션별 snap scroll)
- **히어로**: 미니멀 타이포 (좌측 정렬, 굵은 산세리프, 배경은 어둠→밝음 그라디언트)
- **카드 UI**: 서비스별 카드에 일러스트 + 설명 텍스트 (결제위젯, 브랜드페이, 해외결제, 현금영수증)
- **가맹점 로고 클라우드**: 원형 아이콘으로 메가스터디, 리디북스, 야놀자, 무신사, 쏘카, 클래스101 등 배치
- **수수료 테이블**: 깔끔한 카드형 가격표 (카테고리별 분리)
- **FAQ**: 아코디언 접기/펴기
- **프로세스 스텝**: STEP 1→2→3→4 수평 타임라인
- **컬러**: #0064FF (토스 블루), 화이트 배경, 그레이 텍스트

### 2-3. 네이버페이 커넥트 디자인 분석

- **히어로**: 제품명 "Npay connect" + 상담신청 CTA
- **기능 카드**: 4가지 핵심 기능을 GIF 애니메이션으로 표현
  - 결제: 카드 탭/QR 스캔 모션
  - 리뷰: 결제→리뷰 연결 플로우
  - 마케팅: 쿠폰/적립 자동 제공 모션
  - 홍보: 매장 이벤트 알림 모션
- **상담 폼**: 하단에 즉시 상담 신청 폼 내장 (전화번호 + 매장명)
- **특징**: 모바일 퍼스트, 세로 스크롤 단일 페이지, GIF 중심

### 2-4. 우리편 웹사이트 적용 포인트

**즉시 반영 가능:**
1. ✅ 풀페이지 히어로 (비디오 슬롯 이미 있음)
2. 🔲 제품 카테고리별 GIF/영상 카드 (네이버페이 스타일) → Gemini로 생성
3. 🔲 프로세스 스텝을 수평 타임라인으로 리디자인 (토스페이먼츠 스타일)
4. 🔲 가격/수수료 비교 카드 UI (토스페이먼츠 수수료 테이블 참고)
5. 🔲 하단 상담 폼 직접 내장 (네이버페이 스타일, 현재 플로팅만 있음)
6. 🔲 가맹점/파트너 로고 클라우드를 원형 아이콘으로 업그레이드

**동적 표현 추가:**
- 제품별 GIF 애니메이션 (설치 과정, 단말기 작동, CCTV 화면 등)

## 4. 3팀 합의 사항 — 우리편 핵심 톤앤매너

### 절대 원칙
1. **어미**: "~해요" 친근한 존댓말
2. **문장**: 20자 이내, 핵심만
3. **금지**: 기술 용어 나열, 불안 마케팅, 과장 ("무조건", "100%")
4. **컬러**: #3182F6 (토스 블루) 기반, 화이트 여백 충분
5. **이미지**: 실제 매장 사진 or Gemini 리드로잉 (스톡 이미지 금지)
6. **CTA**: 모든 섹션에 하나씩, "무료", "쉬운" 강조

### 우리편만의 차별점
- 토스/네이버는 **결제** 중심 → 우리편은 **매장 인프라 전체** (인터넷+단말기+CCTV)
- 경쟁사는 단일 제품 → 우리편은 **원스톱 패키지**
- 핵심 메시지: "따로따로 알아보지 마세요. 우리편이 한 번에 해결해드려요."
